# 随心记 更新日志

## v0.5.1 (2026-04-17)

### 回退

- 回退 v0.5.0/v0.6.0/v0.7.0 的 CSV 相关功能
- 删除 utils/csv-import.js、utils/gbk.js
- 删除 pages/csv-import/ 目录
- 移除 app.json 中 csv-import 页面注册
- 移除记账页「CSV导入/AI导入」tab
- 移除 record.js 中 onGoCSVImport 方法
- 记账页恢复为「拍照记账」+「手动记账」两个 tab

### 原因

- 微信导出的是 xlsx 格式，小程序无法直接解析
- 支付宝 CSV 存在 GBK 编码问题
- AI 视觉识别方案更简洁，继续使用现有拍照记账功能即可

## v0.4.0 (2026-04-17)

### 新增

- 新增账单编辑页（bill-edit），支持修改任意一笔账单的所有字段
  - 修改金额、商户、分类、支付渠道
  - 修改级别（日常/额外）
  - 修改归属项目（可将旅游期间的日常支出移出项目）
  - 修改标签
  - 一键「移出项目」按钮（快速将误归入的账单移出）
  - 删除账单
- 首页账单列表点击跳转编辑页
- 项目详情页关联账单点击跳转编辑页

### 修复

- 修复 record.js 中 loadBill 方法仍在使用 wx.cloud.database() 的问题，改为本地存储查询

## v0.3.0 (2026-04-17)

### 修复

- 修复首页和报表页月份切换箭头显示为 `&lt;` 和 `&gt;` 的问题，改用 Unicode 字符 `‹` 和 `›`

### 新增

- 项目类型区分：支持「时间段」和「关键词」两种项目类型
  - 时间段：按起止日期匹配，适合旅游、出差等有明确时间范围的项目
  - 关键词：按商户关键词匹配，适合3D打印机、装修等按类型归集的项目
- 项目创建页增加类型选择卡片（带图标和说明）
- 项目列表和详情页显示项目类型标签（蓝色=时间段，橙色=关键词）
- 关键词类型项目创建时需填写匹配关键词，时间类型需选择日期范围
- 策略模板增加 project_type 字段，标记适用的项目类型

## v0.2.0 (2026-04-17)

### 变更

- 数据存储从微信云开发改为本地存储（wx.setStorageSync）
- 移除云开发依赖（app.js、app.json、project.config.json）
- 移除云函数依赖，所有数据操作改为本地实现
- 项目页删除操作改用 db.deleteProject 替代直接调用云数据库

### 影响

- 无需开通云开发即可运行
- 无需 npm install 安装云函数依赖
- 数据存储在本地，换设备需手动迁移
- 后续可随时切回云开发方案

## v0.1.0 (2026-04-17)

### 新增

- 项目初始化，搭建微信小程序完整骨架
- 首页：月度总览，日常/额外支出概览，饼图占比
- 记账页：
  - 拍照上传账单截图 + AI视觉模型解析
  - 解析结果确认列表（✅已确认/⚠️待确认/❌未识别）
  - 单条账单编辑（商户、金额、分类、渠道、级别、标签）
  - 策略调整（按时间范围/分类/商户关键词/金额区间批量调整）
  - 默认策略模板（旅游专项、装修专项、大额标记、出差期间）
  - 策略预览 + 确认执行
  - 手动记账模式
- 报表页：分类占比柱状图、TOP10支出排行、趋势/对比占位
- 项目页：额外项目管理（创建/查看/关闭/删除），项目关联账单
- 设置页：
  - AI模型配置（支持OpenAI/Claude/通义千问/DeepSeek）
  - API Key + 自定义API地址 + 连接测试
  - 日常基线预算设置
  - 密码保护
  - 支出分类管理（添加/删除）
  - 策略模板查看
- 组件：bill-item（账单列表项）、category-picker（分类选择器）
- 工具：AI解析模块（多提供商适配）、数据库操作模块
- 云函数：parseBill（批量添加 + 策略执行）
- TabBar图标（10个）
